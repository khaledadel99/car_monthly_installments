# BAIC Monthly Payment Calculator

A Streamlit application to calculate monthly car payments for BAIC vehicles.

## Features
- Select from 3 categories (Price determined automatically)
- Interactive sliders and inputs
- Real-time calculation
- Animated UI with Dark Mode styling

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the application:
```bash
streamlit run app.py
```

## Logo
Place a `logo.png` file in `assets/` or the root directory to display a custom logo.
